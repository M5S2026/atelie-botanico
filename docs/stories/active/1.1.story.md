# Story 1.1: Estúdio de Riscos — Espelho Radial e Desenho Livre

**Epic:** 1 — MVP Core Modules
**Status:** Draft
**Prioridade:** Alta

## Contexto

O app já tem Biblioteca de Riscos (65 SVGs) e Projetor Virtual funcionando. Falta o módulo Estúdio de Riscos que permite a usuária criar seus próprios riscos com espelho radial — feature diferenciadora do produto.

## Acceptance Criteria

- [ ] Canvas de desenho livre com touch/mouse (suavização de traço)
- [ ] 5 modos de espelho: vertical, horizontal, 4 eixos, radial 6x, radial 8x
- [ ] Toggle de espelho visível com preview em tempo real
- [ ] Seleção de motivo base (flor, folha, ramo) que gera SVG inicial no canvas
- [ ] Controles: cor do traço, espessura (1-10px), borracha, limpar tudo
- [ ] Undo/redo (mínimo 20 passos)
- [ ] Exportar como PNG (800px grátis, 3000px pago)
- [ ] Escala de exportação ajustável (50%-200%)
- [ ] Botão "Usar no Projetor" que envia o risco criado para o módulo Projetor

## Technical Notes

- Novo componente `src/components/Estudio.jsx`
- Canvas API: paths com `beginPath/moveTo/lineTo/quadraticCurveTo`
- Espelho: duplicar pontos com transformação de coordenadas
- Integrar na tab bar do App.jsx (nova aba entre Riscos e Projetor)
