# Ateliê Botânico

PWA para pintoras decorativas — estúdio de riscos, projetor virtual via câmera, simulador de parede com orçamento.

## Stack
- React + Vite (PWA)
- Supabase (auth, DB, storage)
- Stripe (pagamentos BRL)

## Commands
- `npm run dev` — development
- `npm run build` — production build
- `npm run preview` — preview build

## Current
Scaffold inicial — app base extraído do PRD, biblioteca de SVGs pronta.
